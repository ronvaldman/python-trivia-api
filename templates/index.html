Getting Started...

  • Run the Python file.
  • Select a port to run your local server on (default port is 5000).
  • Consider using Postman for ease of use.

Methods:

  • GET http://localhost:5000/trivia - for all trivia questions
  • GET http://localhost:5000/trivia/random - for a random trivia question
  • GET http://localhost:5000/trivia/420 - for a specific trivia question
  • POST http://localhost:5000/trivia/add - specify a unique ID in JSON (see example below)
  • DELETE http://localhost:5000/trivia/delete/id - for a specific ID
